Infosmog

Ce script permet de chercher les données d'observations pour un mois pour NO2, Pm25,et O3. Il génère un fichier csv et un fichier excel. Le fichier excel est celui qui contient tout les calculs comme moyenne mobile 3h et les maximum des région.

IQA

IQA prend les bulletin du site d'archive de l'Ontario. Il faut télécharger en "unformatted". Ceci parse le text par région et l'ajoute à la fin du fichier CSV.

XML Parser

Le programme prend les fichiers XML et trouve ces données: CASi, CASe, heure émission, endroit, et fuseau horaire. Pour chauque prévision qui contient une CASe, le script va écrire les données dans le fichier CSV.
